bash
<!-- ABGR: Wnpx - grzcbenel olcnff: hfr urnqre "K-Qri-Npprff: lrf" -->

而上面那行明顯是一段加密，分析過後，是 ROT13 加密，然後將他解密

bash
<!-- NOTE: Jack - temporary bypass: use header "X-Dev-Access: yes" -->

他提示我們只要在 Html 的 Header 加上 X-Dev-Access: yes 就可以驗證通過。
